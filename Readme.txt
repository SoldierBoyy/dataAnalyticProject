# Описание

Проект направлен на анализ данных и построение прогнозной модели по данным о погоде в Санкт-Петербурге. Используются различные библиотеки Python для обработки данных, выполнения статистического анализа и построения регрессионных моделей.
В архиве Distribution.zip представлен проект по анализу данных и визуализации данных. Внутри архива 2 папки, одна с файлами для Google Colab, вторая с файлами для визуализации в сервисе Grafana.

# Инструкция по запуску кода

Шаг 1: Вход в Google Colab
Перейти по ссылке Google Colab: https://colab.google/
Войти в свою учетную запись Google, если требуется.

Шаг 2: Загрузка блокнота
В Google Colab нажать на кнопку «Файл» в верхнем левом углу.
Выбрать «Открыть блокнот».
В открывшемся окне выбрать вкладку «Загрузить».
Нажать кнопку «Выберите файл» и выберите файл нужного блокнота (Курсовая.ipynb) на вашем компьютере.

Шаг 3: Загрузка файлов с данными
После открытия блокнота в Colab, на левой панели будет значок папки. Необходимо нажать на него.
В появившемся окне проводника файлов нажать кнопку «Загрузить».
Выбрать csv-файлы с данными (например, "Saint-Petersburg 2022-09-01 to 2024-02-25" и др.) на вашем компьютере и загрузить их.
Убедиться, что загруженные файлы появились в списке файлов в Colab.

Шаг 4: Запуск кода
Проверить, что все необходимые файлы загружены.
Нажать кнопку «Подключиться» в правом верхнем углу экрана для подключения к среде выполнения.
Запустить каждую ячейку блокнота по очереди, нажимая значок Play (треугольник) слева от каждой ячейки или использовать сочетание клавиш Shift + Enter для выполнения ячеек последовательно.
